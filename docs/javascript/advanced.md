# JavaScripti edasijõudnud teemad

::: info Tuleb varsti
See peatükk on praegu koostamisel. Sisu lisatakse peagi.
:::

Selles peatükis käsitleme edasijõudnud JavaScripti teemasid:

- Asünkroonne programmeerimine (`async`/`await`, Promises)
- Destruktureerimine ja spread-operaator
- Moodulid (`import`/`export`)
- Klassid ja prototüübid
- Error handling (`try`/`catch`)
- Funktsionaalse programmeerimise põhimõtted (`map`, `filter`, `reduce`)
