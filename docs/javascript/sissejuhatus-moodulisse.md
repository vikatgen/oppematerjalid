
# M6. Veebirakendused: Javascript

::: info
Antud kursusel ei ole õpiväljundeid. Kursuse mõte on toetada programmeerimise algtõdedega ning Javascripti õppimiseks.
:::

:house: - 60 AT / 30 kohtumist
:calendar: - 70 IT

**Eesmärk** on saada aru ning teada, kuidas kasutada JavaScripti veebirakenduste loomiseks.

##### Kuidas kursus on ülesse seatud?
Kursus on jagatud mitme osa vahel. Esimene osa on programmeerimise algtõed, mille kontrollimiseks tuleb arutelu vormis eksam.
Eksami toimumisaja anname teada vähemalt kuu aega enne, et jõuaksite valmistuda.
Esimese osa raames arutame, kuidas arvuti haldab mälu, mida meil on vaja informatsiooni salvetamiseks, haldamiseks ja kasutamiseks.
Eksami alla kuuluvad programmeerimise alustõed:
- muutujad, skoop, hoisting, omistamine, deklareerimine
- andmetüübid, matemaatilised operaatorid, andmetüüpide sisseehitatud meetodid
- tingimuslaused, tingimuste loomine, operaatorid
- korduslaused, tsükliloogika, tsüklimuutuja kasutamine, tsüklitingimus
- funktsioonid, parameetrid, return võtmesõna

Kui programmeerimise alustõed on selged, alustame Javascripti spetsiifilise osaga (DOM, asünkroonne programmeerimine, sündmused, lubadused jne).
Teise osa hindamine toimub praktilise tööga, kus kasutame Javascripti veebirakenduste loomiseks:
- Sündmuste loomine, reageerimine
- DOM manipuleerimine
- Asünkroonne programmeerimine
- Event loop
- Lubadused (Promise, async/await)
